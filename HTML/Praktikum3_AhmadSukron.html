Bagian 1: Penggunaan Tag HTML Dasar

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat struktur dasar sebuah halaman web. Berikut adalah beberapa tag HTML yang sering digunakan:

  • <h1> hingga <h6>: Tag heading
  • <p>: Tag paragraf
  • <a>: Tag hyperlink
  • <img>: Tag gambar
  • <table>: Tag tabel
  • <ul> dan <li>: Tag list
  • <div>: Tag pembungkus (container)
  • <form>: Tag formulir
